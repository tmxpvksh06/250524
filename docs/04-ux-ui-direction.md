# UX UI Direction

## Current Goal

사용자가 제공하는 Screenshot 폴더의 참고 이미지를 기반으로 사주 플랫폼 전체 UX/UI를 설계합니다.

## Design Scope

초기 범위:

- 메인 화면
- 회원가입 및 로그인
- 카카오 로그인 진입
- 사용자 대시보드
- 상담 이미지 업로드
- 향후 상담 신청 화면
- 모바일 반응형 화면

## Visual Source

사용자 제공 이미지:

- 프로젝트 루트 `Screenshot` 폴더 안의 이미지

AI 생성 이미지:

- Codex 내부 이미지 생성 스킬 사용
- 생성 이미지는 `media/generated`에 정리
- 실제 UI 반영 이미지는 `media/ui`에 정리

## UX Principles

- 첫 화면에서 사주 플랫폼이라는 성격이 즉시 드러나야 합니다.
- 로그인/회원가입 흐름은 단순해야 합니다.
- 모바일에서도 주요 CTA가 명확해야 합니다.
- 상담 신청과 이미지 업로드는 사용자가 망설이지 않도록 단계가 짧아야 합니다.
