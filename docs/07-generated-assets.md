# Generated Assets

Codex 내부 이미지 생성 스킬을 사용해 사주 플랫폼용 이미지를 생성했습니다.

## Workspace Assets

생성 원본 정리:

```text
media/generated/hero-saju-guide.png
media/generated/card-compatibility.png
media/generated/card-wealth.png
media/generated/card-yearly.png
media/generated/card-tarot.png
```

프론트 사용 경로:

```text
frontend/public/images/hero-saju-guide.png
frontend/public/images/card-compatibility.png
frontend/public/images/card-wealth.png
frontend/public/images/card-yearly.png
frontend/public/images/card-tarot.png
```

## Asset Usage

`hero-saju-guide.png`:

- 메인 히어로
- 정통사주 카드

`card-compatibility.png`:

- 사주궁합 카드

`card-wealth.png`:

- 재물운 분석 카드

`card-yearly.png`:

- 2026 신년운 카드

`card-tarot.png`:

- 타로 카드

## Generation Mode

- Mode: built-in `image_gen`
- Style: Korean webtoon/mobile fortune app illustration
- Reference basis: `Screenshot` 폴더의 사주 앱형 카드 이미지
